The pipeline is where opportunities move from discovery into action.

## Use the pipeline for decisions

Move an opportunity into the pipeline when it needs review, assignment, pricing, partner outreach, capture planning, or a bid/no-bid decision.

## Suggested stages

Use stages that match how your team works. A simple starting point:

| Stage | Meaning |
| --- | --- |
| Review | Someone needs to qualify the opportunity. |
| Interested | The opportunity looks relevant but needs more diligence. |
| Pursuing | Your team is actively working it. |
| Submitted | A response has been submitted. |
| Won or Lost | The outcome is known. |

## Keep notes decision-focused

Good pipeline notes explain what changed or what decision is needed. Avoid using pipeline notes as a full document dump when the opportunity details already contain the source material.
